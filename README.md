# franchise-intel-app

Broker-facing web app for the Franchise Intelligence Platform. Takes a candidate buyer profile, returns a ranked list of 12 to 15 franchise brands with persona-aware reasoning.

**Status:** Pre-scaffold. Real build starts in Phase 2 (Week 2 of MVP) once `../franchise-intel/` has ingested enough Wisconsin FDD data to match against. See `CLAUDE.md` for the phased plan and clone targets.
